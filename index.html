Proyectos

GestorEmpleadosX

Una aplicación de gestión de empleados en Java y Swing. Simplifica la administración de recursos humanos con una interfaz fácil de usar y características esenciales para seguimiento de personal, asignación de tareas y evaluación de desempeño. Eficiente, directo al grano y diseñado para facilitar la vida de los gestores de recursos humanos.

AnimeCatalog .NET

Una aplicación de catálogo de animes desarrollada con .NET y WinForms. Disfruta de un catálogo completo de tus animes favoritos con funcionalidades avanzadas y una interfaz atractiva gracias al framework Bunifu. La persistencia de datos mediante JSON asegura una experiencia de usuario fluida y personalizada, con la capacidad de explorar, buscar y descubrir nuevos animes de manera eficiente.
