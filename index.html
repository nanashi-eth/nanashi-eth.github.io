Proyectos

GestorEmpleadosX

Una aplicación Java y Swing para la gestión eficiente de empleados. Simplifica la administración de recursos humanos con una interfaz fácil de usar. Incluye características esenciales para seguimiento, asignación de tareas y evaluación de desempeño.

AnimeCatalog .NET

Catálogo de animes en .NET y WinForms con interfaz atractiva y funciones avanzadas. Persistencia de datos en JSON para una experiencia personalizada y búsqueda eficiente de animes.
